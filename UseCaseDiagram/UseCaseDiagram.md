## Description

Order_Status는 내가 뭘 주문하였는지 확인

Payment_Method는 결제방식 및 수령방식을 선택함

Show_Delivery_Status는 배달현황을 보여는 주는 것

  

방문수령: 주문접수완료 -> 조리중 -> 픽업가능

배달수령: 주문접수완료 -> 조리중 -> 배달 중 -> 배달완료 or 주문취소
